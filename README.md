## 🎯 Features
- 🔐 Secure login & role-based access (Student/Teacher/Admin)
- 📅 Automatic session grouping by level & availability
- ✅ Attendance tracking & progress recording
- 📢 Announcements & notifications
- 📊 Dashboards for all user roles

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Hosting:** Netlify (Prototype) / InfinityFree (Production)
